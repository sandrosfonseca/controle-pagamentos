# Sistema de Controle de Pagamentos

Um sistema simples para controle e gerenciamento de pagamentos, desenvolvido como uma aplicação web estática.

## Funcionalidades

- Cadastro de novos pagamentos
- Visualização de todos os registros
- Exclusão de pagamentos já realizados
- Interface responsiva e fácil de usar

## Como usar

1. Clone este repositório ou faça o download dos arquivos
2. Abra o arquivo `index.html` em seu navegador
3. Comece a gerenciar seus pagamentos!

## Publicação no GitHub Pages

Este projeto pode ser facilmente publicado no GitHub Pages:

1. Crie um repositório no GitHub
2. Faça o push deste código para o repositório
3. Vá em Settings > Pages
4. Selecione a branch main e salve
5. Seu site estará disponível em `https://seu-usuario.github.io/nome-do-repositorio/`

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage para persistência de dados
